# Birthday Mini Story Asset Map

## Use By Scene

All scenes except the Happy Birthday scene rotate their GIF set every 2 seconds. The Happy Birthday scene intentionally keeps one stable birthday GIF so the main reveal feels clean.

### Scene 1 - Secret Lock
- `start.gif` - main soft hug sticker.
- `soft-cats.gif` - gentle couple moment.
- `together.gif` - cute together moment.
- `sleepy-pat.gif` - calm caring moment.
- `please.gif` - cute pleading expression.

### Scene 2 - Suspense
- `please.gif` - quiet waiting.
- `mochi-walk.gif` - simple cute movement.
- `cute-lie.gif` - soft playful pause.
- `start.gif` - warm reassurance.

### Scene 3 - Memories
- `soft-cats.gif` - first smile / softness.
- `together.gif` - missing each other.

### Scene 4 - Countdown

### Scene 6 - Secret Letter

### Scene 7 - Final Surprise
- `hug.gif` - final hug.
- `kiss-clean.gif` - kiss moment.
- `love-clean.gif` - love moment.
- `hearts-clean.gif` - romantic finish.

## Avoid Or Use Carefully

- `love.gif` - has built-in text, only use if the text is acceptable.
- `hug.gif` - may contain text in some frames; acceptable for final only if it does not distract.
- Any GIF with unrelated text like "On the bus", "Go tera go", "YAY", or foreign phrases should be avoided in core romantic scenes.

## Rule Of Thumb

Use GIFs without built-in captions for the main emotional scenes. Let HTML render the important words, because then the project feels personal instead of like a reused sticker pack.
